# Image Prompts — codingal

All images in the build are the client's real assets pulled from codingal.com (course art, student projects, teacher cluster, dashboard). They are decent quality and on-brand, so the build ships with real visuals. Prompts below are optional upgrades for the team when they want to elevate specific slots.

---

## Hero image (flagged for upgrade)

**Why:** The hero currently reuses the instructor hexagon cluster (`teachers.png`). It works, but a dedicated "child coding live, mid-delight" shot would land the "writes their first program in week one" promise harder, in line with the Edukids / CodeQuest references.

```
Image: Hero visual
Prompt: Photorealistic, a happy 9-year-old child at a laptop in a bright home, looking at the screen with a small proud smile, a simple colourful code or game project visible on the screen, warm natural window light, soft coral and cream tones in the background, shallow depth of field, candid not posed, 4K
Negative: no stock-photo stiffness, no fake grin, no text overlays, no watermarks, no dark or corporate office
```

## Student project thumbnails (optional)

**Why:** Current project images are real student screenshots (keep them). If the team wants a consistent frame, generate matching coloured backplates rather than replacing the work itself.

```
Image: Project card backplate
Prompt: Flat soft-rounded coloured panel, single accent colour (coral / sky blue / grass green / sunshine yellow), subtle paper texture, gentle drop shadow, empty centre for a screenshot, playful kids-education feel
Negative: no gradients that fight the screenshot, no clip-art, no text
```

## Class preview (optional)

**Why:** `dashboard.png` shows the real dashboard, which is the right proof. If a live-class photo is wanted alongside it:

```
Image: Live 1-on-1 class moment
Prompt: Photorealistic over-the-shoulder view of a child on a video call with a friendly instructor visible on screen, code editor open in a second window, cosy home desk, warm light, authentic and candid, 4K
Negative: no posed studio look, no headset cliche, no watermarks
```
