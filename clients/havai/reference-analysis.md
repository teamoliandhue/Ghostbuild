# Reference Analysis — havinic.webflow.io

A Webflow e-commerce template built for home decor / designer furniture. Adaptable to Havai's fan/cooler catalogue.

## Visual language
- **Palette (template)**: Off-white bg `#fafafa`, near-black text `#0e1011`, soft pink accent `#ffdede`, neutral greys `#758696` `#5d6c7b`. Generally light/airy/premium.
- **Fonts**: Outfit (display + body) and DM Sans (secondary). Modern geometric sans-serifs, no serifs anywhere.
- **Rhythm**: Generous whitespace, big margins, headings in title case, body text centred in hero sections.
- **Motion**: Heavy use of fade-in-on-scroll (every h2 starts at `opacity:0` and animates in). Subtle, never distracting.

## Section order (this becomes Havai's homepage skeleton)
1. **Mini top bar** — promo strip: "Transform Your Space – Save 30% Today!" (we'll swap for Havai-relevant copy)
2. **Navbar** — logo left, menu centre (Home / About / Shop / Blogs / Contact), cart icon right
3. **Hero** — large image + bold h1 "Modern Living with Designer Furniture" + descriptor line + CTA
4. **Categories** — animated reveal grid of product categories
5. **About Us** — image + text split, h2 "About Us" + paragraph
6. **Discover more** / featured section — "Premium quality designs created for your home" with imagery
7. **Services / Why Us** — "Best services that we have" — service tiles on dark bg
8. **Latest Products** — product grid (4 across), each with image, name, price, add to cart
9. **Latest Blogs** — 3-up blog card grid
10. **Newsletter** — "Stay Connected with the World of Home Style" + email input + "We're just a call away" contact card
11. **Footer** — dark, multi-column

## How we adapt for Havai
- Replace home-decor headline with cooling/fan-led copy (the content audit will write this)
- Swap product categories grid to: Air Coolers, Commercial Mist Fans, BLDC Pedestal/Wall, BLDC Ceiling Fans, Tower Fans, Exhaust Fans, Chairs, Iron Boards
- Replace "Latest Blogs" with "Real cooling results" testimonial grid (Havai has none on site — strong upgrade opportunity)
- Keep newsletter section but reframe as "Get installation guides + warranty alerts"
- Apply Havai colour: navy #465299 as the accent (CTAs, links), keep off-white bg from reference, use yellow #ffb503 sparingly for sale/promo badges only
- Font: keep the reference's geometric sans feel — Outfit or Inter, depending on team confirmation

## What we lift wholesale from the reference
- Light off-white background + near-black text contrast
- Fade-in-on-scroll for every section heading
- Title-case h2s, centred hero text
- Generous section padding, no cramped sections
- Dark footer with grouped link columns + newsletter
